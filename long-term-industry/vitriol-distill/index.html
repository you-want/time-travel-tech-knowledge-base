土法蒸馏硫酸与硝酸¶

基础元数据¶

  • 分类：长期工业
  • 落地难度：完整产业链（数年）
  • 必备原料：绿矾（硫酸亚铁）、硝石（硝酸钾）、黄铁矿
  • 是否需要特殊工具：是（蒸馏装置、冷凝管、耐酸容器）
  • 适用场景：官府建功、高收益
  • 标签：#高收益 #官府建功 #完整产业链

重要声明¶

⚠️ 本页面内容仅供历史技术研究和网文创作参考。硫酸和硝酸均为强腐蚀性危险化学品，操作不当会造成严重伤害。请在合法合规的前提下进行研究。切勿在现实生活中尝试制作。

一、现代底层原理（简化）¶

硫酸和硝酸被称为"化工之母" — 几乎所有化工生产过程都离不开它们。

  • 硫酸：很重、很黏的酸，能溶解很多金属，能吸水
  • 硝酸：很活泼的酸，能溶解金、铂以外的几乎所有金属

在古代条件下，这两种酸可以通过干馏法（在密闭容器中高温加热矿物，收集挥发出的气体）来制备。

二、古代实操完整步骤¶

第一部分：制备硫酸¶

原料：绿矾（硫酸亚铁，FeSO₄·7H₂O）¶

绿矾是铜绿（蓝绿色的矿物），在很多地方的山上都能找到，也可以从铁锈水中结晶得到。

蒸馏装置¶

    ┌─────────┐
    │  冷凝   │ ← 通冷水
    └────┬────┘
         │
    ═════╪═════ ← 收集硫酸液滴
         │
    ┌────▼────┐
    │  蒸馏   │ ← 高温加热
    │  瓶     │
    └─────────┘

制作步骤¶

  1. 准备蒸馏器：
  2. 找一个耐热的陶瓷罐（坩埚）
  3. 罐口用泥封住，中间插一根长导管

  4. 导管的另一端伸入另一个收集瓶中

  5. 装料：

  6. 将绿矾碾碎，放入蒸馏罐

  7. 装到罐的 1/3 满即可

  8. 加热：

  9. 用木炭火将蒸馏罐加热到 500-600°C
  10. 保持恒温，不能忽高忽低

  11. 加热时间：6-12 小时

  12. 收集：

  13. 绿矾受热分解，释放出三氧化硫气体
  14. 气体通过导管进入收集瓶
  15. 在收集瓶中与水蒸气结合，形成硫酸

  16. 最初收集的是稀硫酸，需要进一步浓缩

  17. 浓缩：

  18. 将稀硫酸倒入陶瓷皿中
  19. 在太阳下暴晒或用文火慢慢蒸发水分
  20. 直到液体变得黏稠如油（"矾油"）

第二部分：制备硝酸¶

原料：硝石（硝酸钾，KNO₃）+ 绿矾 + 硫酸¶

  1. 将硝石烘干后研碎
  2. 与等量的绿矾粉末混合
  3. 加入少量水调成糊状
  4. 放入蒸馏器中加热
  5. 硝石受热分解，释放出二氧化氮气体（红棕色）
  6. 气体通入水中，形成硝酸

更简单的方法（直接加热硝石与硫酸）： 1. 将浓硫酸慢慢倒入研碎的硝石中 2. 混合物放入蒸馏器 3. 加热，收集挥发出的硝酸蒸汽 4. 冷凝后得到硝酸

三、优势对比（对比无酸化工）¶

  1. 溶解金属：硫酸和硝酸能溶解几乎所有金属，提取金属元素
  2. 面料处理：硝酸可以处理丝绸，使其更柔软有光泽
  3. 炸药原料：硝酸是制作黑火药改进型炸药的原料
  4. 冶金辅助：酸洗可以去除金属表面的氧化物
  5. 化工基础：有了酸，才能做更精细的化工产品

四、局限性与短板¶

  1. 极度危险：
  2. 硫酸和硝酸都腐蚀性极强
  3. 接触皮肤会造成严重烧伤
  4. 蒸汽刺激呼吸道
  5. 必须在通风极好的地方操作
  6. 设备要求高：
  7. 蒸馏器必须是陶瓷或玻璃（金属会被腐蚀）
  8. 收集瓶也必须耐酸
  9. 密封材料不能用橡胶（古代也没有橡胶）
  10. 产量极低：一次蒸馏只能得到几十毫升酸
  11. 纯度不高：土法制出的酸含有杂质，不适合精密用途
  12. 时间成本：整个蒸馏过程需要 12-24 小时

  • 酸是蒸馏化工的基础 → 本页面
