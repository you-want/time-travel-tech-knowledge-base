# 铁器淬火回火工艺

## 基础元数据
- 分类：冶金锻造
- 落地难度：1个月可量产
- 必备原料：铁器、水、油、木炭火
- 是否需要特殊工具：是（火炉、淬火槽）
- 适用场景：官府建功、高收益
- 标签：#高收益 #官府建功

## 一、现代底层原理（简化）

铁器是软还是硬，不只是看铁好不好，更要看怎么处理。同样的铁块，经过不同的加热和冷却方式，硬度可以差几倍。

**淬火**就是把铁器烧到通红（约 800°C），然后迅速浸入冷水或冷油中急速冷却。这样铁的内部结构会发生变化，变得非常硬 — 但同时也非常脆，像玻璃一样一敲就碎。

**回火**就是把淬火后过硬过脆的铁器重新加热到较低的温度（约 200-400°C），然后让它在空气中慢慢冷却。这样能去掉一部分"脆劲"，让铁器既硬又有韧性 — 砍得动东西但不会断。

打个比方：淬火是让铁"绷紧神经"，回火是让铁"放松一点但还不松懈"。

## 二、古代实操完整步骤

### 第一步：淬火

#### 1. 加热

1. 将铁器放在木炭火炉中加热
2. 用皮囊或风箱鼓风，让炉温达到 750-850°C
3. **判断温度的方法**（没有温度计）：
   - 暗红色 → 约 600°C（不够）
   - 樱桃红 → 约 750°C（偏低）
   - 橙红色 → 约 800°C（刚好）
   - 亮橙色 → 约 900°C（偏高）
   - 黄白色 → 1000°C+（太热了）
4. 保持铁器颜色为橙红色约 10-15 分钟，让整件铁器内外温度均匀

#### 2. 淬火冷却

1. 准备好淬火介质：
   - **水淬**：适合碳钢，冷却速度快，硬度高但容易裂
   - **油淬**：适合合金钢，冷却速度适中，不易裂
   - **盐水淬**：比纯水冷却更快，硬度更高
2. 用铁钳夹住烧红的铁器
3. **迅速**（1-2 秒内）完全浸入淬火液中
4. 保持浸没状态直到不再冒气泡（约 30 秒）
5. 取出后用布擦干

### 第二步：回火

#### 1. 重新加热

1. 将淬火后的铁器放在炉子上
2. 用较小的火（约 300°C）慢慢加热
3. **判断回火温度的方法** — 观察铁器表面的氧化膜颜色：

| 表面颜色 | 温度 | 硬度 | 韧性 | 适用工具 |
|----------|------|------|------|----------|
| 淡黄色 | 220°C | 很高 | 低 | 刀具、钻头 |
| 紫色 | 280°C | 高 | 中高 | 斧头、凿子 |
| 蓝色 | 300°C | 中高 | 高 | 弹簧、锤子 |
| 深蓝色 | 350°C | 中 | 很高 | 铁锤、铁砧 |

4. 达到目标颜色后，让铁器在空气中自然冷却（不要水冷！）

#### 2. 检验

1. **砍木测试**：用铁器砍一段拇指粗的木头
   - 如果铁器变形 → 回火过度（太软了），需要重新淬火
   - 如果铁器崩口 → 回火不足（太脆了），需要更高温度回火
   - 如果铁器完好无损 → 热处理成功

2. **弯曲测试**（适用于薄铁片）：
   - 将铁器弯成 U 形
   - 不回弹也不断裂 = 合格
   - 一弯就断 = 太脆，需要重新回火
   - 弯了不恢复 = 太软，需要重新淬火

## 三、优势对比（对比未经热处理的铁器）

1. **硬度提升**：淬火后的铁器硬度可提高 2-3 倍
2. **使用寿命延长**： properly 回火的铁器不容易卷刃或断裂
3. **工具多样化**：不同工具需要不同的硬度和韧性组合
4. **军事价值**：淬火刀剑在冷兵器时代具有压倒性优势
5. **经济效益**：一块普通铁经过热处理后价值翻倍

## 四、局限性与短板

1. **经验依赖**：温度全靠眼看，新手经常练手（废铁）
2. **开裂风险**：淬火时如果冷却太快，铁器可能直接裂开
3. **变形风险**：加热不均匀会导致铁器弯曲
4. **反复处理会疲劳**：同一件铁器淬火回火次数太多会变脆
5. **不是所有铁都适合**：含碳量太低（熟铁）淬火效果不明显，含碳量太高（生铁）容易裂
