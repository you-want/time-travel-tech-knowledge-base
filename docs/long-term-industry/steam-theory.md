## 一、现代底层原理（简化）

蒸汽机的原理听起来很简单：

1. 把水烧开，变成蒸汽
2. 蒸汽体积膨胀约 1600 倍，产生压力
3. 用这个压力推动活塞运动
4. 活塞的往复运动变成旋转运动，带动机器

**但做起来极其困难。** 因为蒸汽机的核心不是"原理"，而是三个工程难题：

### 难题一：密封

活塞要在气缸里来回运动，但气缸和活塞之间不能漏气。漏气了就没压力，没压力就没动力。要密封好，间隙必须控制在 0.1 毫米以内 — 这需要一个精密的车床来加工。

### 难题二：耐压

锅炉里的蒸汽压力可达几个到十几个大气压。如果锅炉不够结实，就会爆炸 — 而且是很厉害的爆炸。古代的铁锅和陶罐根本承受不了。

### 难题三：效率

即使做出了蒸汽机，效率也很低 — 只有 1%-2% 的热能变成了机械能。也就是说，烧 100 斤木炭，只有 1-2 斤的力气被用上了。其余的都浪费掉了。

## 二、古代可以尝试的"蒸汽实验"

虽然真正的蒸汽机造不出来，但以下简单的蒸汽实验是可以做的：

### 实验一：火箭原理演示（ Aeolian 汽球）

1. 找一个金属试管或小锅
2. 装入少量水
3. 用木炭火加热
4. 水沸腾后，蒸汽从管口喷出
5. **现象**：试管会向后跳动 — 这就是反冲力原理

### 实验二：抽水装置

1. 做一个密封的钢瓶（厚壁铁罐）
2. 瓶内装一半水
3. 加热瓶身（或在瓶内加热水）
4. 蒸汽压力把水压出一根管子
5. **用途**：可以把水提到比水源高的地方

### 实验三：简单的蒸汽泵（纽科门机的简化版）

这是历史上第一台实用的蒸汽机（1712 年），原理如下：

1. 在一个大气缸中注入少量水
2. 加热水变成蒸汽，把气缸中的空气挤出去
3. 停止加热，注入冷水让气缸迅速冷却
4. 蒸汽遇冷变成水，体积急剧缩小，气缸内形成真空
5. 大气压力把活塞压下去
6. 活塞带动杠杆，可以抽水

**局限**：这个机器效率极低（<1%），但可以用来抽水。需要精密的气缸和活塞密封 — 古代很难做到。

## 三、为什么蒸汽机是工业革命的门槛

蒸汽机之所以被称为"工业革命的心脏"，是因为它是**第一个把热能转化为机械能的装置**。有了它：

- 工厂不再需要建在河边（不需要水力）
- 矿山可以抽出深层地下水（不需要人工舀水）
- 火车和轮船成为可能（不需要马匹）

但蒸汽机的前提条件是：

1. **精密加工能力** — 车床、铣床、刨床
2. **合格的金属材料** — 能耐压、耐高温
3. **可靠的密封技术** — 活塞环、垫片
4. **压力计和安全阀** — 防止爆炸

这四项中，前三项全部属于[精密机械制造](../limits/high-precision.md)的范畴，在古代条件下**无法实现**。

## 四、古代能做的"蒸汽替代方案"

| 需求 | 蒸汽机方案 | 古代替代方案 |
|------|-----------|-------------|
| 抽水 | 蒸汽泵 | 水车、人力畜力泵 |
| 传动 | 蒸汽机+皮带 | 水车+木齿轮 |
| 加热 | 蒸汽加热管道 | 火道、烟道 |
| 动力 | 蒸汽机+飞轮 | 水车+曲柄连杆 |

**结论**：在有水源的地方，水车是蒸汽机最好的替代品。水车不需要精密加工，不需要耐压容器，不需要密封 — 木头就能搞定。

## 五、局限性与短板

1. **精密加工是硬伤** — 没有车床就做不出合格的气缸
2. **材料是硬伤** — 古代钢铁强度不够，锅炉会爆炸
3. **密封是硬伤** — 皮革和麻绳密封撑不住蒸汽压力
4. **效率太低** — 即使能做出来，费柴费水，不划算
5. **危险性极高** — 低压锅炉也可能爆炸，伤及操作者
