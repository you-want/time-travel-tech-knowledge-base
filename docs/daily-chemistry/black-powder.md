## 一、现代底层原理（简化）

火药是什么？简单说就是**硝石、硫磺、木炭三者按比例混合后，点燃会发生极速燃烧**的粉末。

- 硝石是"氧气搬运工" — 它在受热时会放出氧气
- 硫磺和木炭是"燃料" — 它们在有氧气的情况下快速燃烧
- 三者缺一不可：只有木炭和硫磺，烧得不旺；只有硝石和木炭，点不着；只有硝石，什么都不是

古代炼丹家偶然发现了这个组合，从此改变了世界。

## 二、古代实操完整步骤

### 第一步：准备原料

#### 1. 硝石（硝酸钾）

**土法提硝**：
1. 找一面南朝北的墙根（常年有硝霜析出的地方），刮下墙皮表面的土
2. 或者收集厕所旁边/猪圈下面的泥土（含氨氮多，容易生成硝石）
3. 将土放入锅中，加入大量清水煮沸
4. 过滤后，将滤液倒入浅盘中静置
5. 几天后盘底会出现白色针状晶体 — 这就是粗硝石
6. 重复溶解-结晶 2-3 次提高纯度

**更简单的方法**：直接从市场上购买提纯过的硝石

#### 2. 硫磺

- 从硫磺矿开采（多在温泉附近）
- 或从市场上购买（中药店有售）
- 粉碎后用煤油溶解过滤，除去杂质，重新结晶得到较纯的硫磺

#### 3. 木炭

- 选质地柔软的树木（如柳木、枣木）
- 在密闭的炭窑中低温烧制（不完全燃烧）
- 得到的木炭要干燥、轻脆
- 研磨成细粉

### 第二步：配比

**标准火药配比（质量比）**：

| 原料 | 比例 | 说明 |
|------|------|------|
| 硝石 | 75% | 越高爆发力越强 |
| 硫磺 | 10% | 降低着火点 |
| 木炭 | 15% | 提供燃料 |

这是最经典的"一硝二磺三木炭"比例（按重量：硝石 15 份、硫磺 2 份、木炭 3 份）。

**用途调整**：
- 爆破用：硝石提高到 80%，硫磺降到 8%
- 烟花用：降低硝石到 60%，增加木炭到 25%，硫磺 15%
- 信号弹用：硝石 70%，增加木炭到 20%

### 第三步：研磨与混合

1. **分别研磨**：将三种原料各自研磨成极细的粉末（越细越好）
2. **过筛**：用细筛（80 目以上）筛三次，确保颗粒均匀
3. **混合**：在干燥的环境中，将三种粉末按比例混合
4. **搅拌均匀**：用木铲轻轻翻拌至少 30 分钟
5. **潮解压制**（可选）：加少量水使粉末微潮，压成小块烘干。这样运输更安全，使用时掰碎即可

### 第四步：测试

1. 取一小撮火药放在石板上
2. 用点燃的香靠近
3. 正常情况应瞬间燃烧并发出明亮的火焰
4. 如果燃烧缓慢，说明硝石比例不够
5. 如果只冒烟不燃烧，可能是木炭太粗

## 三、安全规范（至关重要！）

### 制作过程中的安全

| 禁忌 | 原因 |
|------|------|
| **严禁明火** | 火药粉尘遇火星即爆 |
| **严禁金属工具** | 碰撞产生火花 |
| **严禁穿化纤衣物** | 静电可能引爆 |
| **必须在通风处** | 有毒气体排出 |
| **每次少量制作** | 减少事故损失 |
| **远离居民区** | 万一出事不伤及无辜 |

### 储存安全

1. 存放在干燥、阴凉、通风的地方
2. 用陶罐或木箱密封储存
3. 远离火源、水源（潮解失效）
4. 标明"火药"字样，专人看管
5. 每次取用后立即密封

### 使用安全

1. 点火后立刻撤离到安全距离
2. 未爆的火药不要立刻靠近，等待至少 15 分钟
3. 受潮的火药不要暴晒后继续使用（可能失效或燃烧不稳）
4. 废弃火药用水浸湿后深埋

## 四、优势对比（对比古代武器）

1. **杀伤力飞跃**：火铳的威力远超弓箭和投石机
2. **训练成本低**：一个农夫训练三天就能用火铳，而弓箭需要多年练习
3. **心理威慑**：巨大的声响和烟雾对未经训练的敌军有极强的震慑效果
4. **攻城利器**：火药可以炸毁城墙，古代城防形同虚设

## 五、局限性与短板

1. **威力有限**：黑火药的爆炸力远不如现代炸药（TNT），只能用于燃烧和轻度爆破
2. **发射不稳定**：每批次火药的质量差异大，射程和威力不稳定
3. **武器配套**：需要坚固的火铳/火炮才能承受火药爆炸力，对[冶金技术](../metallurgy-forging/blast-furnace-improve.md)有要求
4. **烟太大**：燃烧产生大量白烟，遮蔽视线，打完一炮什么都看不见
5. **法律风险**：私自制造火药在古代属于重罪，需要获得官府许可
6. **储存危险**：火药易燃易爆，储存不当会自燃或被盗
