# 纺车升级、植物染色与丝麻加工

## 基础元数据
- 分类：纺织印染
- 落地难度：当日落地（手搓纺线）/ 1个月可量产（纺车、染色）
- 必备原料：棉花、麻类、植物染料、蚕丝
- 是否需要特殊工具：是（纺车、染缸、织布机）
- 适用场景：生存致富、高收益
- 标签：#低成本手工 #高收益 #无电力 #民生

## 一、现代底层原理（简化）

古人穿衣靠两种材料：**麻**（平民）和**丝**（贵族）。棉花是后来才传入中国的，但一旦有了棉纺技术，保暖性和舒适度都远超麻布。

纺织的核心流程就三步：
1. **取纤维** — 从植物（棉花、麻）或动物（蚕）身上拿到长长的纤维
2. **纺成线** — 把短纤维搓成长线，线的强度决定了布的强度
3. **织成布** — 经纬线交错编织，加上染色，就成了衣服

其中**纺车**是最大的生产力飞跃 — 手搓一天纺半斤线，用纺车一天能纺五六斤。而**染色**则是从"能穿"到"穿得好"的关键 — 好的染色技术能让布料价值翻十倍。

## 二、古代实操完整步骤

### 第一部分：纺车升级（手搓 → 纺车）

#### 手摇纺车（单锭）

这是最基础的纺车，原理和辘轳类似 — 用轮子的转速放大手的动作。

**建造步骤**：

1. **制作纺轮**：找一块圆形的木头（直径 30-50 厘米），中间钻孔，装上轴。这就是纺轮（也叫锭盘），越重惯性越大，转得越稳。
2. **支架**：用两根竖木夹住纺轮轴的上下两端，做成一个"门"字形支架。
3. **脚踏板**（可选）：用绳子或皮带连着踏板，脚踩踏板带动纺轮转动，双手解放出来操作纤维。
4. **纱锭**：在纺轮上方装一根细木棍（锭杆），顶端挂钩子用来绕线。
5. **使用方法**：
   - 一手拉纤维（让纤维保持连续不断）
   - 一手或脚踏带动纺轮转动
   - 纤维被捻成线，绕在锭杆上

**效率对比**：
| 方式 | 日产（棉线） | 说明 |
|------|-------------|------|
| 手搓捻线 | 0.25 斤 | 手指来回搓 |
| 单锭纺车 | 0.5-1 斤 | 效率提升 2-4 倍 |
| 多锭纺车（3-4 锭） | 2-3 斤 | 需要熟练工人 |

#### 脚踏双锭纺车（进阶版）

1. 在支架上装两根锭杆
2. 用脚踏板驱动主轴，双手各操作一根锭杆
3. 效率比单锭再翻倍
4. 元代黄道婆改进的纺车就是这种

### 第二部分：棉花加工（从棉籽到棉絮）

#### 去籽（弹棉）

1. **手工敲打**（最原始）：把带籽的棉花摊在板上，用木棍敲打，把棉籽震出来
2. ** Roller 轧花机**（进阶）：
   - 两根木辊平行安装，间距可调
   - 把带籽棉花夹在中间滚动
   - 棉花从辊间穿出，棉籽被拦住
   - 效率比手打高 10 倍以上

#### 弹松

1. 用弓弦琴式的"弹弓"（一根弯弓+一根弦）
2. 弦上涂松脂增加粘性
3. 拨动弦，震动波把棉絮弹松
4. 弹松后的棉花才能纺成均匀的线

### 第三部分：麻类加工（葛、麻、苎麻）

麻比棉花难处理，因为麻纤维在茎秆里，需要先"脱胶"才能取出来。

**沤麻法**：

1. 砍下麻杆（苎麻或大麻杆均可）
2. 剥去外皮，得到麻皮
3. 把麻皮泡在河水或水塘中 7-15 天（夏天快，冬天慢）
4. 泡到外皮发臭、黏液脱落
5. 捞出后敲打、搓洗，麻纤维就出来了
6. 晾干后就可以纺线了

**关键点**：沤的时间太短脱胶不彻底，太长纤维会烂。看水面冒小气泡、麻皮能和内秆轻松分离就是刚好。

### 第四部分：植物染色（最常见的染料）

古代染料主要来自植物，每种颜色对应不同的原料：

| 颜色 | 染料来源 | 媒染剂 | 说明 |
|------|----------|--------|------|
| 黄色 | 黄檗（黄柏）、栀子、槐花 | 无 | 最易染，直接上色 |
| 蓝色 | 蓼蓝（板蓝根叶子） | 石灰水 | 需要发酵还原，稍复杂 |
| 红色 | 茜草、苏木、红花 | 明矾 | 茜草根是最传统的红色染料 |
| 黑色 | 橡碗（栎树果壳）、五倍子 | 铁浆 | 铁浆让颜色变深黑 |
| 紫色 | 紫草 | 明矾 | 紫色在古代是高贵颜色 |
| 绿色 | 先用蓝染再用黄染 | 明矾 | 套染法：先蓝后黄 = 绿 |
| 粉色 | 红花（少量） | 明矾 | 红花是最高贵的淡色染料 |

#### 染色步骤：

1. **预处理**（退浆）：布料先用热水煮一遍，去掉织布时上的浆（让布更吸色）
2. **媒染**：
   - 把布料先泡在媒染剂（明矾、铁浆、石灰水）中 30 分钟
   - 媒染剂像"胶水"，帮助染料固定在纤维上
   - 不同媒染剂会让同一种染料呈现不同颜色（比如茜草+明矾=红，茜草+铁浆=紫黑）
3. **浸染**：
   - 把处理好的布料放入染缸
   - 保持 60-80°C 的文火，煮 30-60 分钟
   - 不断翻动，确保上色均匀
4. **氧化**：
   - 取出布料，在空气中展开晾 15-30 分钟
   - 很多染料（尤其是蓝色）离开染液后会继续氧化变深
5. **清洗**：用清水漂洗 2-3 遍，去掉浮色
6. **晾干**：阴干（不要在烈日下晒，会褪色）

**套染法**（一种染出多种颜色）：

- 先染蓝 → 再染黄 = 绿色
- 先染红 → 再染蓝 = 紫色
- 先染黄 → 再染红 = 橙色
- 先染蓝 → 再染红 = 褐色

### 第五部分：织布机（从手工编织到织机）

#### 腰机（最原始）

1. 经线一端绑在树上或柱子上
2. 另一端系在织者的腰带上
3. 织者坐下往后仰，拉紧经线
4. 用手穿梭纬线
5. 用木综框分经线（上下交替）
6. 用木梭子穿纬线
7. 用筘打紧纬线

**效率**：每天约织 5-10 厘米布，极慢。

#### 脚踏提花织机（进阶）

1. 用脚踏板控制综片升降（解放双手）
2. 可以织出更宽的布（幅宽可达 50 厘米以上）
3. 可以织出简单的花纹（斜纹、缎纹）
4. 效率比腰机高 3-5 倍

## 三、优势对比（对比古代现有技术）

1. **产量提升**：纺车比手搓效率高 4-10 倍，一个人一天的产量抵得上三四个人
2. **布的质量**：纺车纺出的线更均匀，织出来的布更结实漂亮
3. **染色多样**：植物染色让衣服从"能穿"变成"好看"，价值翻 10 倍
4. **棉布普及**：有了棉纺技术，保暖衣物不再是奢侈品
5. **可量产出售**：染好的布、纺好的线都可以拿到集市上卖

---

## 四、局限性与短板

1. **纺车需要木工手艺**：制作纺轮、支架、锭杆需要一定的木工经验
2. **染料获取**：某些染料（如红花、紫草）不是到处都有，需要贸易
3. **套染复杂**：先蓝后绿、先红后紫，每种套染都要单独控制温度和時間
4. **染色牢度**：土法染色的布洗几次会褪色，不如现代化学染料持久
5. **丝的生产门槛高**：养蚕需要桑树、控制温度、防病，比棉麻复杂得多
6. **织机笨重**：大型织机需要固定安装，不能随身携带
