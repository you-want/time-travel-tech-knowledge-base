# 草木灰提取纯碱与烧碱

## 基础元数据
- 分类：日用化工
- 落地难度：1个月可量产
- 必备原料：草木灰、石灰、动物脂肪/油脂
- 是否需要特殊工具：是（陶罐、铁锅、滤布）
- 适用场景：生存致富、高收益
- 标签：#低成本手工 #高收益 #无电力

---

## 一、现代底层原理（简化）

纯碱（碳酸钠）和烧碱（氢氧化钠）是化工之母 — 做肥皂需要它，漂洗衣物需要它，后续很多化工反应也需要它。

草木灰里本来就有碳酸钾（植物燃烧后残留），把它溶解在水里，过滤出来，再把石灰（氧化钙）加进去，就能把碳酸钾转化成氢氧化钠（烧碱）。烧碱溶液放在敞口容器里让空气慢慢接触，又会变成碳酸钠（纯碱）。

简单说：**草木灰 → 灰汁（含碳酸钾） → 加石灰 → 烧碱 → 放空气中 → 纯碱**

## 二、古代实操完整步骤

### 第一步：制备草木灰滤液

1. 收集干净的草木灰（最好是硬木燃烧后的灰，含碱量最高）
2. 将草木灰装入底部有小孔的木桶（"漏盆"）
3. 从上方缓慢注入清水
4. 收集从底部渗出的液体，这就是"灰汁"（含碳酸钾）
5. 浓缩灰汁：将灰汁倒入大铁锅，用文火慢慢熬煮，水分蒸发后得到白色或灰黑色粉末（粗制碳酸钾）

### 第二步：制备烧碱（氢氧化钠）

1. 准备生石灰（将石灰石煅烧得到）
2. 将生石灰加入少量水中，它会自己崩解成熟石灰（氢氧化钙）
3. 把熟石灰加入灰汁中，搅拌均匀
4. 此时发生化学反应：碳酸钾 + 氢氧化钙 → 氢氧化钠 + 碳酸钙沉淀
5. 静置数小时，底部的白色沉淀是碳酸钙，上面的清液就是烧碱溶液
6. 小心地将清液倒出（不要搅起底部的沉淀）
7. 将清液倒入铁锅熬煮，得到白色固体 — 这就是烧碱（氢氧化钠）

**注意**：烧碱腐蚀性极强！操作时必须戴手套（厚布手套），避免接触皮肤和眼睛。

### 第三步：制备纯碱（碳酸钠）

方法一（空气碳化法，较慢但安全）：
1. 将烧碱溶液放在浅口的陶瓷盘中
2. 暴露在空气中，让二氧化碳慢慢溶入
3. 几周后，溶液表面的氢氧化钠会变成碳酸钠
4. 蒸发结晶，得到白色纯碱粉末

方法二（苏打法，较快但有风险）：
1. 将木灰（富含碳酸氢钾）加入烧碱溶液中
2. 搅拌后静置，碳酸钾与烧碱反应生成碳酸钠
3. 过滤、蒸发、结晶

### 第四步：提纯（可选）

1. 将粗制的碱溶解在少量热水中
2. 过滤掉不溶性杂质
3. 重新蒸发结晶
4. 得到的白色晶体纯度更高

## 三、优势对比（对比古代现有技术）

1. **自用+出售**：自己做肥皂用不完的可以拿到集市上卖，利润丰厚
2. **替代进口**：优质纯碱在古代属于紧俏货，自产自用省去买钱
3. **化工基础**：有了碱，才能做后续的很多化工（如[制皂](soap-make.md)、[糖精](sugar-refine.md)）
4. **原料无限**：草木灰烧火就有，石灰到处都有，几乎零成本
5. **用途广泛**：洗衣、制皂、食品加工（如做粽子用的碱水）、腌制

## 四、局限性与短板

1. **烧碱危险**：腐蚀皮肤和眼睛，操作不当会受伤
2. **纯度不高**：土法生产的碱含有杂质（氯化钾、碳酸钾等），不适合精细化工
3. **产量有限**：每次只能熬一小锅，规模化生产需要多个工同时操作
4. **能耗高**：熬煮灰汁和碱液需要大量柴火
5. **气味**：熬煮过程中有刺激性气味，需要在通风处操作
