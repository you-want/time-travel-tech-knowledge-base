# 电力与芯片相关全部作废技术

## 基础元数据
- 分类：不可实现技术
- 落地难度：永远无法实现
- 必备原料：需要稀有金属、高纯硅、绝缘材料等古代不存在之物
- 是否需要特殊工具：是（需要精密制造体系）
- 适用场景：避坑参考 — 不要在这些技术上浪费时间
- 标签：#避坑 #无电力 #不可实现

---

## 一、为什么这些技术全部不可行

电力技术的本质是**控制电子在导体中的定向流动**。要实现这一点，至少需要三个条件：

1. **稳定的电源** — 发电机（需要磁场+线圈+转动）、电池（需要精确化学配方）
2. **导线与绝缘体** — 高度纯净的金属拉丝 + 橡胶/塑料绝缘层
3. **精密元件** — 电阻、电容、晶体管等，全部需要微米级甚至纳米级加工

古代能有的材料（铜、铁、木头、布、油脂）无法满足上述任何一个条件。哪怕你知道发电机的原理，没有高纯度铜线、没有精密轴承、没有绝缘材料，也造不出来。

## 二、逐一分析：哪些做不到，为什么

### 1. 发电机 / 电动机

- **常见误区**："用磁铁在线圈里转就能发电"
- **现实问题**：
  - 天然磁石（磁铁矿）磁力太弱且无法塑形
  - 电磁铁需要大量铜线和稳定电流 — 先有鸡还是先有蛋
  - 机械转动需要精密轴承，古代车床做不到
  - 即使勉强转动，产生的电量微乎其微，连点亮一盏小油灯都不够

### 2. 电池

- **常见误区**："铜锌盐水就是电池"
- **现实问题**：[简易铜锌电池](../long-term-industry/volta-battery.md)确实能产生微弱电流，但：
  - 电压极低（约 0.76V 每节）
  - 电流极小，带不动任何实际负载
  - 要串联成可用电池组，需要大量高纯度金属和精确电解液
  - 结论：只能做物理实验演示，不能当能源用

### 3. 电灯 / 照明

- **常见误区**："做 filament（灯丝）就能发光"
- **现实问题**：
  - 钨丝熔点 3410°C，古代最高炉温约 1500°C
  - 玻璃抽成真空灯泡需要精密玻璃工艺
  - 即使做出灯丝，没有稳定电源也无法持续使用
  - **替代方案**：改进油灯（用更精炼的植物油 + 细棉芯）即可显著提升亮度

### 4. 电报 / 通信

- **常见误区**："摩尔斯电码很简单"
- **现实问题**：
  - 电报机需要电磁继电器、精密开关
  - 长距离通信需要电线杆 + 千米级铜线
  - 信号衰减需要放大器 — 而放大器又是电子管/晶体管
  - **替代方案**：烽火台、驿站、旗语是古代最可靠的远程通信方式

### 5. 半导体 / 芯片

- **常见误区**："硅不就是沙子吗，很容易搞吧"
- **现实问题**：
  - 芯片用的是**99.9999999%** 纯度的单晶硅（9 个 9）
  - 光刻精度达到纳米级别（头发丝的十万分之一）
  - 需要超净间（空气中一粒灰尘就能毁掉整批芯片）
  - 需要数百种特种气体、光刻胶、蚀刻液
  - 结论：**这是人类工业文明的巅峰之作，古代绝对不可能**

### 6. 无线电 / 广播

- **常见误区**："马可尼早就发明无线电了"
- **现实问题**：
  - 真空管需要精确抽真空 + 高纯度钨丝 + 精密玻璃密封
  - 调谐电路需要可变电容器（精密加工）
  - 天线需要千米级金属线
  - 即使收到微弱信号，也没有放大和解码手段

### 7. 机械计算 / 早期计算机

- **常见误区**："巴贝奇分析机不就是齿轮吗"
- **现实问题**：
  - 19 世纪的精密齿轮加工需要专用机床
  - 即使做出原型，存储和处理能力极其有限
  - 没有电信号输入输出，实用性为零

## 三、古代真正能做的"电学实验"

虽然电力技术不可行，但以下简单电学现象可以体验：

| 现象 | 方法 | 效果 |
|------|------|------|
| 摩擦起电 | 琥珀摩擦羊毛布 | 吸引纸屑，可做趣味演示 |
| 静电火花 | 莱顿瓶（玻璃罐+内外铝箔+盐水） | 产生可见电火花，震撼但无用 |
| 原电池 | [铜锌盐水电池](../long-term-industry/volta-battery.md) | 微弱电流，仅能刺激舌头 |

这些实验可以作为"展示知识"的噱头，但**不能作为实用技术**。

## 四、局限性与短板

1. **不要在这些技术上浪费任何时间** — 从原理到量产之间隔着整个工业革命，不是一个人能跨越的
2. **警惕"差不多就行"的心态** — 电学对精度要求极高，差一点就完全没用
3. **别人不信你的时候** — 拿出摩擦起电的小把戏比讲电磁感应理论管用一百倍
