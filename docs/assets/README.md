# 图片资源说明

此目录用于存放知识库相关的图片资源，包括：

- 工艺流程图
- 结构示意图
- 对比照片
- 地图和示意图

## 目录结构

```
assets/
├── agriculture/    # 农业相关图片
├── chemical/       # 化工相关图片
└── metallurgy/     # 冶金相关图片
```

## 命名规范

- 文件名使用英文小写 + 连字符，如 `compost-pile-diagram.png`
- 图片格式优先使用 PNG（示意图）和 JPG（照片）
- 最大宽度 1200px，文件大小不超过 500KB
