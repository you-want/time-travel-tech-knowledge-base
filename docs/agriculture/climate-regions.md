## 一、现代底层原理（简化）

同样的种子，在南方一年能收三季，在北方一年只能收一季。同样的堆肥方法，在湿热地区一个月就好，在寒冷地区半年都发酵不了。

这不是技术有问题，而是**气候决定了技术的适用性**。穿越到古代，首先要搞清楚自己到了什么地方，然后调整技术方案。

气候的核心变量就四个：**温度、降水、光照、土壤**。每个变量都会影响哪些技术能用、怎么用。

## 二、中国主要气候带划分

### 第一带：热带（华南沿海、海南、云南南部）

| 特征 | 说明 |
|------|------|
| 年均温 | 22°C 以上，全年无冬 |
| 降水 | 1500-2500mm，雨季旱季分明 |
| 生长期 | 全年可种植，一年三熟 |
| 代表地区 | 广州、海口、西双版纳 |

**适合的技术**：
- 水稻一年可种 2-3 季（早稻 + 晚稻 + 冬作）
- 热带作物：椰子、香蕉、菠萝、甘蔗
- 堆肥发酵极快（30°C+环境下 2-3 周即可完成）
- 病虫害全年活跃，需要持续防虫 → [土法防虫](../agriculture/pest-control.md)
- 雨水多，排水要好 → [简易水利灌溉](../agriculture/irrigation.md)（侧重排水而非灌溉）

**特别注意**：热带地区湿度大，粮食和工具容易发霉，需要做好防潮。

---

### 第二带：亚热带（长江流域及以南）

| 特征 | 说明 |
|------|------|
| 年均温 | 15-22°C |
| 降水 | 1000-1500mm |
| 生长期 | 250-300 天，一年两熟 |
| 代表地区 | 南京、武汉、杭州、成都 |

**适合的技术**：
- 稻麦轮作（夏天种稻、冬天种麦）
- 柑橘类水果（橙子、橘子、柚子）
- 茶叶种植
- 桑树养蚕 → [纺织技术改良](../daily-chemistry/textile-dye.md)
- 堆肥发酵较快（夏季 1 个月，冬季 2-3 个月）
- 冬季温和，可以种越冬作物（蚕豆、油菜）

**特别注意**：长江流域梅雨季节（6-7月）连续阴雨，要注意谷物防潮和水利排涝。

---

### 第三带：温带（华北、关中、中原）

| 特征 | 说明 |
|------|------|
| 年均温 | 10-15°C |
| 降水 | 500-800mm，集中在夏季 |
| 生长期 | 180-220 天，一年一熟或两年三熟 |
| 代表地区 | 北京、西安、郑州、济南 |

**适合的技术**：
- 旱作为主：小麦、玉米、高粱、谷子（小米）
- 冬季寒冷，不能种越冬水稻
- 堆肥发酵中等速度（夏季 1 个月，冬季基本停滞）
- 需要灌溉系统 → [简易水利灌溉](../agriculture/irrigation.md)（北方缺水，灌溉更重要）
- 冬季需要保温：温室可用（玻璃/纸覆盖的暖棚，古代已有）
- 土壤偏碱性，堆肥有助于改良

**特别注意**：北方冬季长达 4-5 个月，必须储备足够的过冬粮食和蔬菜。

---

### 第四带：干旱/半干旱区（西北、内蒙古）

| 特征 | 说明 |
|------|------|
| 年均温 | 5-15°C（温差极大） |
| 降水 | < 400mm，很多地方 < 200mm |
| 生长期 | 120-180 天 |
| 代表地区 | 兰州、乌鲁木齐、银川 |

**适合的技术**：
- **坎儿井**（地下暗渠引水）— 新疆特有的节水灌溉方式，减少蒸发
- 绿洲农业 — 只在有水的地方种植，周围是大片荒漠
- 耐旱作物：粟（小米）、高粱、莜麦、土豆
- 畜牧业比重高 — 羊、骆驼、牛
- 堆肥发酵快（干燥+高温），但缺水是最大限制
- [土法防水水泥](../architecture-mechanic/ancient-concrete.md) — 用于修建水渠

**特别注意**：干旱区最大的问题是水，不是技术。任何农业技术都必须围绕"节水"展开。

---

### 第五带：高原气候（青藏高原、云贵高原）

| 特征 | 说明 |
|------|------|
| 年均温 | 0-15°C（海拔越高越冷） |
| 降水 |  varies widely |
| 生长期 | 60-150 天（海拔决定） |
| 代表地区 | 拉萨、昆明 |

**适合的技术**：
- 青稞 — 唯一能在高寒地区成熟的主粮
- 温室种植困难（气温太低）
- 畜牧业为主（牦牛、藏羊）
- 日照强烈 — 紫外线强，但光合作用也强
- 堆肥发酵慢（低温），需要加厚保温

## 三、不同气候带的技术调整对照表

### 堆肥发酵速度

| 气候带 | 夏季发酵时间 | 冬季发酵时间 | 调整方法 |
|--------|------------|------------|----------|
| 热带 | 2-3 周 | 1-2 个月 | 冬季加盖保温 |
| 亚热带 | 1 个月 | 2-3 个月 | 常规操作 |
| 温带 | 1 个月 | 基本停滞 | 冬季停堆，春季重启 |
| 干旱区 | 3-4 周 | 2-3 个月 | 加水是关键，覆盖保湿 |
| 高原 | 1.5 个月 | 基本停滞 | 加厚堆体保温 |

### 作物选择

| 作物 | 热带 | 亚热带 | 温带 | 干旱区 | 高原 |
|------|------|--------|------|--------|------|
| 水稻 | ✅ 三季 | ✅ 两季 | ❌ | ❌ | ❌ |
| 小麦 | ❌ | ✅ 越冬 | ✅ 冬/春 | ✅ 春麦 | ❌ |
| 玉米 | ✅ 两季 | ✅ 两季 | ✅ 一季 | ✅ | ❌ |
| 粟(小米) | ❌ | ❌ | ✅ | ✅ | ❌ |
| 土豆 | ❌ | ✅ | ✅ | ✅ | ✅ |
| 青稞 | ❌ | ❌ | ❌ | ❌ | ✅ |
| 甘蔗 | ✅ | ✅ | ❌ | ❌ | ❌ |
| 棉花 | ✅ | ✅ | ✅ | ✅ | ❌ |

### 灌溉需求

| 气候带 | 灌溉需求 | 推荐方式 |
|--------|---------|----------|
| 热带 | 旱季需灌溉，雨季不需 | 水渠 + 蓄水池 |
| 亚热带 | 总体不需大量灌溉 | 排水为主 |
| 温带 | 春季和初夏需灌溉 | 井灌 + 水渠 |
| 干旱区 | 全程依赖灌溉 | 坎儿井 + 滴灌（土法） |
| 高原 | 夏季降水足够 | 融雪水渠 |

## 四、因地制宜的原则总结

1. **先看气候，再选技术** — 同样的技术在不同地方效果可能差 10 倍
2. **顺天时应地利** — 不要跟气候对抗，而是利用它
3. **本地经验最可靠** — 当地农民种了几百年的作物，就是最好的参考
4. **跨区域引种要谨慎** — 南方的水稻种到北方可能不抽穗，北方的麦子在南方容易倒伏
5. **复合利用** — 一个地区可以同时发展多种技术（如南方稻鱼共生、北方桑基鱼塘）

## 五、局限性与短板

1. **气候数据不可得**：古代没有气象站，只能通过观察自然现象判断（蚂蚁搬家预示下雨、燕子低飞要下雨等）
2. **极端天气无法预测**：旱灾、洪灾、霜冻来临前很难察觉
3. **土壤差异**：同一气候带内，土壤类型也可能完全不同
4. **海拔影响**：山区气候垂直变化大，"一山有四季，十里不同天"
