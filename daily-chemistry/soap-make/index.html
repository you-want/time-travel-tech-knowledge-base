油脂与草木灰土法制皂¶

基础元数据¶

  • 分类：日用化工
  • 落地难度：1个月可量产
  • 必备原料：动物油脂（猪油、牛油）、草木灰（或纯碱）、食盐
  • 是否需要特殊工具：是（大铁锅、木模具、布滤）
  • 适用场景：生存致富、官府建功
  • 标签：#低成本手工 #高收益 #无电力

一、现代底层原理（简化）¶

肥皂是什么？简单来说就是油脂和碱相遇后变身的东西。

油脂（猪油、牛油、植物油）本身是滑溜溜的，洗不干净手上的油污。碱（草木灰水里就有）能让油脂发生一种叫"皂化"的变化 — 变身后的一端喜欢水、一端喜欢油。这样洗手的时候，喜欢油的那一头抓住手上的油污，喜欢水的那一头被你用水一冲，油污就被带走了。

一句话总结：肥皂是油和碱的"孩子"，天生既能抓油又能亲水。

二、古代实操完整步骤¶

第一步：准备原料¶

获取油脂¶

  • 动物油脂（推荐）：猪油、牛油、羊油最容易获得
  • 植物油：菜籽油、花生油、麻油也可以用，但皂化速度较慢
  • 油脂可以先用文火加热融化，去掉杂质（肉渣、血水）

获取碱液¶

有两种方法：

方法 A — 草木灰滤液（传统方法）： 1. 收集燃烧后的草木灰（越纯越好，不要混入泥土） 2. 将草木灰放入一个底部扎了几个小孔的木桶中 3. 从上面缓慢倒水，水会从桶底慢慢渗出 4. 渗出的水就是碱液（古人叫"灰汁"） 5. 测试浓度：把鸡蛋放进去，如果鸡蛋浮出水面约一元硬币大小，浓度刚好

方法 B — 购买纯碱/烧碱： - 如果当地有制碱工坊，可以直接购买（见草木灰提取纯碱）

第二步：皂化反应¶

  1. 将油脂倒入大铁锅中，文火加热到约 60°C（不沸腾，表面微微起泡）
  2. 缓慢倒入碱液，边倒边用木棍不停搅拌
  3. 保持文火，持续搅拌约 1-2 小时
  4. 判断皂化完成：用木棍挑起一点混合物，滴入冷水中。如果在水面上凝固成柔软的团，说明皂化完成了；如果散开或溶解，还需要继续煮

第三步：盐析与成型¶

  1. 皂化完成后，往锅里撒入大量食盐（或倒入浓盐水）
  2. 继续搅拌，此时混合物会分成两层：
  3. 上层：白色的肥皂膏（皂基）
  4. 下层：浑浊的废水（废液，含有甘油和杂质）
  5. 用勺子把上层的肥皂膏捞出
  6. 倒入木模具中（可以是.any 有凹槽的木盒子），摊平
  7. 冷却凝固后取出，切成块状
  8. 放在通风处晾晒 1-2 周，肥皂会变硬，更好用

三、优势对比（对比古代现有技术）¶

  1. 清洁效果远超草木灰水：肥皂能洗掉手上的油污，灰汁只能洗掉灰尘
  2. 不伤手：正确使用的话，肥皂的碱性温和，不会像纯碱那样伤手
  3. 可量产出售：生产成本极低（油脂和草木灰都便宜），售价可以有较高利润
  4. 卫生改善：定期用肥皂洗手，能大幅减少皮肤病和肠道传染病
  5. 多功能：不仅可以洗澡洗手，还可以洗衣服、洗碗

四、局限性与短板¶

  1. 油脂消耗：做 1 公斤肥皂大约需要 1 公斤油脂 + 适量碱液，油脂成本占大头
  2. 时间较长：从皂化到成型到晾晒，全程需要 2-3 周
  3. 碱性控制：碱液浓度不好掌握，太浓伤手，太稀洗不干净
  4. 硬水问题：在泉水硬的地方（钙镁离子多），肥皂会产生"皂垢"，效果打折
  5. 气味：动物油脂做的肥皂有膻味，可以用香料掩盖

五、拓展延伸配套技术¶

  • 碱从哪里来 → 草木灰提取纯碱
  • 油脂从哪来 → 堆肥养更多牲畜
  • 净水保障 → 明矾净水工艺（洗肥皂需要干净的水）
  • 白糖提纯 → 白糖脱色提纯（副产品糖蜜可以用来做肥皂添加剂）
